This repository contains [BRLC Token](https://github.com/cloudwalk/brlc-token) swaps pool smart contracts.

## Build and test

``` sh
# Install all dependencies
npm install

# Compile all contracts
npx hardhat compile

# Run all tests
npx hardhat test
```
